# TreasureHunt Plugin

## 概要

宝探しゲーム


## コマンド

- ```/treasurehunt start```|```/treasurehunt end```

    ゲームを開始、終了するコマンド


- ```/treasurehunt add``` | ```/treasurehunt remove```

    ゲームの得点の対象となるアイテムを追加、削除するコマンド  


- ```/treasurehunt config gameTime```
    
    ゲームの制限時間を設定するコマンド  
    最小値 -> 10

- ```/treasurehunt show```

    ゲームの得点の対象として追加されたアイテムを表示する
### 対象
Minecraft Java Edition 1.19.3
